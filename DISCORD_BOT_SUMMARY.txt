================================================================================
GETTING STARTED
================================================================================

1. PREREQUISITES
   - Node.js >= 18.0.0
   - Discord bot token from https://discord.com/developers
   - Bot added to Discord server with proper permissions

2. INSTALLATION
   npm install discord.js dotenv node-schedule

3. CONFIGURATION
   cp integrations/discord/.env.example integrations/discord/.env
   # Edit .env with DISCORD_TOKEN and CLIENT_ID

4. RUN
   node integrations/discord/bot.js

5. TEST
   In Discord: !fortune
   Expected: Beautiful quote embed

================================================================================
PRODUCTION DEPLOYMENT OPTIONS
================================================================================

DOCKER:
  docker build -t dev-fortune-bot -f integrations/discord/Dockerfile .
  docker run --env-file integrations/discord/.env dev-fortune-bot

SYSTEMD:
  Create /etc/systemd/user/dev-fortune-bot.service
  systemctl --user enable dev-fortune-bot
  systemctl --user start dev-fortune-bot

KUBERNETES:
  Deploy with configmap for .env
  One pod running "node integrations/discord/bot.js"
