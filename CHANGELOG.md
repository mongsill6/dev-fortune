# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Community contribution system for quote submissions
- Contributor leaderboard and recognition system
- Quote validation and moderation workflow
- Integration with GitHub Issues for community feedback
- User profiles and submission history tracking

## [0.11.0] - 2026-03-27

### Added
- Slack bot integration for sharing daily fortunes
- Discord bot integration with interactive components
- Webhook module for custom integrations
- Bot command handlers and event listeners
- Rate limiting and security features for bots

## [0.10.0] - 2026-03-27

### Added
- Terminal TUI interactive mode using Ink
- Interactive quote browsing with keyboard navigation
- Real-time search and filtering in terminal
- Beautiful terminal UI with color support
- Save/bookmark functionality in TUI mode

## [0.9.0] - 2026-03-27

### Added
- Docker containerization with multi-stage builds
- Docker Compose configuration for full-stack deployment
- Container health checks and volume management
- Docker Hub image publishing workflow
- Environment-based configuration for containerized deployments

## [0.8.0] - 2026-03-27

### Added
- GitHub Actions CI/CD pipeline
- Automated testing on multiple Node.js versions
- Continuous deployment to npm registry
- Code coverage reporting
- Community contribution templates (PR, Issue, Bug Report)
- Contributing guidelines and code of conduct

## [0.7.0] - 2026-03-27

### Added
- React web frontend with Vite
- Tailwind CSS styling and responsive design
- Quote display with category filtering
- Dark/light theme toggle
- Favorites management and local storage persistence
- Share functionality for social media

## [0.6.0] - 2026-03-27

### Added
- Express.js REST API server
- Swagger/OpenAPI documentation
- RESTful endpoints for quotes, categories, and search
- CORS and error handling middleware
- JSON response formatting
- API rate limiting

## [0.5.0] - 2026-03-27

### Added
- Jest test suite with 74 tests
- Unit tests for quote database module
- Integration tests for CLI functionality
- API endpoint testing
- Test coverage reporting
- Pre-commit test validation

## [0.4.0] - 2026-03-27

### Added
- NPM package structure and configuration
- ESM (ES6 modules) migration from CommonJS
- Package.json metadata and dependencies
- npm registry publication support
- Semantic versioning enforcement
- Build and distribution pipeline

## [0.3.0] - 2026-03-27

### Added
- Multilingual quote database (Korean, Japanese, English)
- Category system for quote organization (motivation, wisdom, humor, etc.)
- Language selection command-line option
- Category filtering and browsing
- Localization support framework
- Translation management utilities

## [0.2.0] - 2026-03-27

### Added
- Complete rewrite of CLI in Node.js
- Commander.js framework for command parsing
- Improved command structure and help documentation
- Color-coded output with chalk
- Better error handling and validation
- Configuration file support

### Changed
- Migrated from bash script to Node.js runtime
- Simplified command syntax and usage patterns

## [0.1.0] - 2026-03-26

### Added
- Initial dev-fortune CLI tool
- Bash script implementation
- 20 curated quotes database
- Basic command-line interface
- Simple quote display functionality
- Random quote selection
